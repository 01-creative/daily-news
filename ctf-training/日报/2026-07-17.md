📄 DREA: Decoupled Reasoning and Exploration Agents for Repository-Level Vulnerability Detection
Sun & Meng 提出双智能体协作架构：规划智能体形成漏洞假设并指导调查方向，探索智能体（轻量本地模型）按需检索仓库级上下文。与单函数分析或固定规则方案不同，DREA 实现了假设驱动的自适应上下文获取，跨三种 LLM 将漏洞检测准确率从 19–26% 提升至 30–42%，同时将 API 成本降低 16–48 倍。作者还发现 26–55% 的真阳性存在"答案正确但推理错误"的现象，揭示了 LLM 漏洞检测中推理质量的瓶颈。
→ https://arxiv.org/abs/2607.13439

📄 ProfMalPlus: Agent-Coordinated Detection of Malicious NPM Packages via Static-Dynamic Analysis Synergy
黄奕恒等提出多智能体协同的 NPM 恶意包检测系统，结合对象敏感行为图与 LLM 推理，通过本地法官智能体独立评估代码切片、全局法官综合裁决的自一致性机制降低误判。对不确定案例自动路由至动态沙箱执行或第三方注册表语义富化。系统 F1 达 98.1%，超越现有检测器 3.5–52.6%，并已在 NPM 中发现 597 个此前未知的恶意包（已被官方移除）。该方法对防范供应链攻击具有直接实用价值。
