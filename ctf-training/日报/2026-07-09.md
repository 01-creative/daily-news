📰 工信部 / CNNVD 预警：Claude Code 存在安全后门，阿里已内部禁用
工业和信息化部通过国家信息安全漏洞库（CNNVD/NVDB）发布风险提示，称 Anthropic AI 编程工具 Claude Code（2.1.91 至 2.1.196 版本）内置监控机制，可在未获用户同意的情况下，通过检测系统时区与代理配置识别位于中国的用户，并将地理位置、身份标识等信息以隐写标记方式嵌入系统提示词回传至 Anthropic 服务器。Anthropic Claude Code 工程师公开承认该机制为今年 3 月启动的"实验"，目的是防止账户转售与竞对模型蒸馏，相关代码已于 7 月 1 日发布的 2.1.198 版本中移除。阿里巴巴此前已于 7 月 3 日将 Claude Code 列为高风险软件，要求全体员工自 7 月 10 日起转向自研工具 Qoder。该事件凸显 AI 开发工具链的供应链安全风险——高校研究团队及实验室应及时审查所使用的 AI 编码工具版本，确保升级至最新安全版本。
